
Za dodavanje u trenutnoj grani:
add .
commit -m "neka poruka"
git push

git pull - dobavljanje izmena

git branch - pregled svih grana

git branch imeGrane - kreiranje nove grane

git checkout imeGrane - prelazak na granu sa datim imenom

iz sporedne grane git push za slanje grane na github

nakon git push iz sporedne grane, na github-u se pojavljuje git pull request
 za spajanje sa glavnom granom (merge pull request) gde se ostavlja komentar sta je uradjeno

git merge master(ili bilo koja grana koja je iznad) - spajanje sa glavnom granom ili granom iznad
